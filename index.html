Chat Application (Long-Polling) - Fullstack

Live Demo | GitHub Repo

This project is a simple real-time chat application built using the long polling technique. It allows users to send and receive messages with a live-like experience, without using WebSockets or any third-party libraries. The backend holds pending client requests and responds only when a new message arrives or a timeout occurs. This simulates a real-time connection and ensures users see new messages almost instantly.

Technologies: HTML, CSS, JavaScript, Node.js, Express.js

Shared Bookmars - Frontend

Live Demo | GitHub Repo

A user-friendly web app that allows switching between two users to view and manage their bookmarks. The app displays bookmarks in reverse chronological order, showing each with a clickable title, description, and creation date. Users can easily add new bookmarks via a form specifying URL, title, and description, which instantly updates the list for the selected user. If no bookmarks exist, a clear message informs the user. Built with accessibility in mind, the app scores a perfect 100 on Lighthouse accessibility tests. The project also includes unit tests to ensure reliable functionality.

Technologies: HTML, CSS, Vanilla JavaScript
